# Rainforest
Misadventures with creating a multi-BVH system in GameMaker
The name comes from the fact that a forest has multiple trees; this system *also* has multiple trees

Primarily learning as I go here, mainly just want a dumping ground for all this ^^;
I owe a lot to Erin Catto's great insight on optimized collision, and just in general parts of [Box2D](https://github.com/erincatto/box2d), which I referenced often during the making of all this

Still a WIP; PLANS.txt shows mainly what I'm working towards, as I generally want something malleable that can be used for fast, arbitrary collisions *not tied to any object* in GameMaker games
